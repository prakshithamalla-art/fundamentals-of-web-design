CSS Box Model Explained

1. Content

This is where your actual content (text, images) appears.

Properties: width, height

Example: The text you're reading now is the content.

2. Padding

Space between content and border. Transparent area inside the box.

Properties: padding, padding-top, padding-right, padding-bottom, padding-left

Example: padding: 20px; creates space around content.

3. Border

A line surrounding the padding and content.

Properties: border, border-width, border-style, border-color

Example: border: 2px solid black; creates a visible boundary.

4. Margin

Space outside the border. Creates distance between elements.

Properties: margin, margin-top, margin-right, margin-bottom, margin-left

Example: margin: 20px; creates space between this box and others.

    background: white;    /* Content background */

}

.button {
    padding: 10px 20px;   /* Space inside button */
    margin: 10px 0;       /* Space around button */
    border: none;         /* Remove default border */
    background: #3498db;  /* Button color */
