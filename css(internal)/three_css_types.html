Three Ways to Apply CSS

1. Inline CSS

This paragraph uses INLINE CSS directly in the style attribute. It has highest priority and overrides other styles.

2. Internal CSS

This content is styled using INTERNAL CSS defined in the style tag in head section.

Internal CSS is good for single-page specific styles.

3. External CSS

This content would be styled by EXTERNAL CSS from style.css file.

External CSS is best for multiple pages and reusability.
